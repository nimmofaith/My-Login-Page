# Login System (React + Flask)

This is a simple authentication system with a React frontend and Flask backend.

## Features
- User Registration
- User Login
- Password validation
- API authentication

## Technologies
Frontend:
- React
- HTML
- CSS
- JavaScript

Backend:
- Flask
- Flask-SQLAlchemy
- SQLite

Deployment:
- Netlify (Frontend)
- Render (Backend)
